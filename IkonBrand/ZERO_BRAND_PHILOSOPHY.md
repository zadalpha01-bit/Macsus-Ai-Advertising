## Filosofi Nama

### Kenapa "ZERO"?

Zero memiliki **dua sisi** yang saling melengkapi:

```
    ┌─────────────────────────────────────┐
    │                                     │
    │   0 = AWAL          0 = AKHIR      │
    │   Pembukaan         Penutupan      │
    │   Kemungkinan       Kenyataan      │
    │   Mimpi             Hasil          │
    │   Nol               Penuh          │
    │                                     │
    └─────────────────────────────────────┘
```

- **Sisi pertama — Awal:** Zero adalah titik di mana segalanya dimulai. Sebelum ada sesuatu, ada nol. Nol adalah ruang kosong yang penuh kemungkinan. Setiap project, setiap ide, setiap mimpi dimulai dari zero.

- **Sisi kedua — Akhir:** Zero juga adalah penutup siklus. Ketika sesuatu selesai, kita kembali ke zero — bukan untuk berhenti, tapi untuk memulai siklus baru dengan pembelajaran yang lebih baik.

**ZERO = Siklus tanpa akhir yang selalu kembali ke awal yang lebih kuat.**

## Core Values

### 1. Keberanian *(Bravery)*

> *"Zero Fear."*

Memulai dari nol butuh keberanian. Tidak ada yang memberi jaminan. Tidak ada yang mengenal. Tapi dengan keberanian, zero bisa menjadi apa saja.

**Dalam praktik:**
- Berani coba hal baru tanpa takut gagal
- Berani beda dari yang lain
- Berani mulai meski belum siap sepenuhnya

---

### 2. Kesederhanaan *(Simplicity)*

> *"Zero Clutter."*

ZERO percaya pada kekuatan kesederhanaan. Tidak perlu rumit untuk menjadi bermakna. Yang penting adalah esensi — bukan noise.

**Dalam praktik:**
- Desain yang clean, tidak norak
- Fitur yang fungsional, bukan gimmick
- Kode yang rapi, performa yang cepat
- Komunikasi yang jelas, tidak bertele-tele

---

### 3. Inovasi *(Innovation)*

> *"Zero Limits."*

Zero tidak punya batas. Karena nol tidak memiliki nilai tetap — ia bisa menjadi berapa saja. Begitu juga brand ini: selalu berevolusi, selalu mencari cara baru.

**Dalam praktikik:**
- Teknologi terbaru (PWA, AI, Web Modern)
- Pendekatan kreatif untuk masalah konvensional
- Terus belajar, terus berkembang
- Tidak puas dengan "cukup"

## Siklus ZERO

```
        ┌──────────────────┐
        │                  │
        ▼                  │
    ┌────────┐        ┌────────┐
    │  ZERO  │───────▶│  IDEA  │
    │ (Awal) │        │        │
    └────────┘        └────────┘
                           │
                           ▼
        ┌──────────────────┐
        │                  │
        ▼                  │
    ┌────────┐        ┌────────┐
    │  DONE  │◀───────│ BUILD  │
    │ (Hasil)│        │        │
    └────────┘        └────────┘
        │
        │  Kembali ke ZERO
        │  untuk siklus baru
        └──────────────────▶ [BACK TO ZERO]
```

**Setiap proyek dimulai dari ZERO → IDE → BUILD → DONE → kembali ke ZERO.**

Tidak ada akhir. Hanya siklus yang terus berputar, terus membaik.
